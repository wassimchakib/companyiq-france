# CompanyIQ France

> **Explorez et analysez les entreprises françaises en un coup d'œil.**

---

## 🚀 Aperçu

> CompanyIQ France est une application front-end réactive permettant de rechercher, visualiser et filtrer les données des entreprises françaises (SIREN). Idéale pour les analystes, investisseurs et enquêteurs souhaitant accéder rapidement à des informations structurées et des visualisations interactives.

**Fonctionnalités clés :**

* Recherche en temps réel par nom d’entreprise ou SIREN

---

## 🛠 Technologies

> **Framework** : React + Vite
> **CSS** : Tailwind CSS + shadcn/ui

---

## 📦 Installation

```bash
# Clone le repo
git clone https://github.com/wassimchakib/companyiq-france.git
cd companyiq-france

# Installe les dépendances
npm install

# Lance le serveur de développement
npm run dev
```

> Ouvrez ensuite votre navigateur sur `http://localhost:5173`.

## 🎯 Workflow Git / GitHub

1. **Branches** :

   * `main`: code stable, déployable
   * `dev`: intégration continue
   * `feature/xxx`: nouvelles fonctionnalités
   * `hotfix/xxx`: corrections urgentes

2. **Commits** : Utilisez des messages clairs et sémantiques :

   * `feat:` pour nouvelles fonctionnalités
   * `fix:` pour corrections de bugs
   * `chore:` pour modifications sans impact fonctionnel

3. **Pull Requests** :

   * Ouvrez une PR depuis `feature/*` vers `dev` ou `main`
   * Décrivez le contexte, captures d’écran et tests réalisés
   * Sollicitez une revue de code avant merge

4. **CI/CD** (optionnel) :

   * Configurez GitHub Actions pour lint, tests et build
